# Changelog

All notable changes to this project will be documented in this file.
The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - 2026-03-17

### Added
- Initial release
- Displays a rotating fullscreen background slideshow from a local directory
- Recursive directory scan; supports jpg, jpeg, png, gif, and webp
- Shuffled playback order
- Configurable change interval and fade transition duration
- Designed for use with MMM-ThemeSwitcher via a `current` symlink that switches between photo sets
